🧠 1. 중앙처리장치 (CPU) 및 연산 장치 (1~25)
Q1.컴퓨터의 두뇌에 해당하며 제어, 연산, 레지스터 등의 코어로 구성된 핵심 반도체 소자는?
👁️ 정답 보기 중앙처리장치 (CPU)
Q2.CPU 내부에서 명령어를 해독하고 각 장치에 제어 신호를 보내 지시하는 장치는?
👁️ 정답 보기 제어장치 (CU, Control Unit)
Q3.CPU 내부에서 산술 연산(+, -, *, /)과 논리 연산(AND, OR, NOT)을 수행하는 장치는?
👁️ 정답 보기 산술논리연산장치 (ALU)
Q4.CPU 내부에 존재하는 초고속 임시 저장 메모리로, 연산의 중간 결과나 명령어 주소를 저장하는 것은?
👁️ 정답 보기 레지스터 (Register)
Q5.CPU가 다음에 실행할 명령어의 주소(기억장소 번지)를 보관하는 레지스터는?
👁️ 정답 보기 프로그램 카운터 (PC)

👁️ 정답 보기 명령 레지스터 (IR)
Q7.연산 장치(ALU)의 입력 데이터를 기억하거나 연산 결과를 일시적으로 저장하는 가장 대표적인 레지스터는?
👁️ 정답 보기 누산기 (Accumulator)
Q8.연산 결과가 양수인지, 음수인지, 0인지 또는 오버플로우가 발생했는지 등의 상태 정보를 저장하는 레지스터는?
👁️ 정답 보기 상태 레지스터 (Status Register)
Q9.컴퓨터 부품들이 서로 데이터를 주고받는 통로를 뜻하는 용어는?
👁️ 정답 보기 버스 (Bus)
Q10.버스의 종류 중 CPU가 메모리나 주변기기에 데이터의 위치를 지정할 때 사용하는 단방향 버스는?
👁️ 정답 보기 주소 버스 (Address Bus)
Q11.버스의 종류 중 CPU, 기억장치, 주변기기 간에 실제 데이터를 주고받는 양방향 버스는?
👁️ 정답 보기 데이터 버스 (Data Bus)
Q12.CPU 속도를 나타내는 단위 중 1초당 100만 개의 명령어를 처리할 수 있는 성능 단위는?
👁️ 정답 보기 MIPS

👁️ 정답 보기 실행 주기 (Execute Cycle)

👁️ 정답 보기 RISC

👁️ 정답 보기 CISC

👁️ 정답 보기 코어 (Core)
Q18.하나의 물리적인 코어를 소프트웨어적으로 두 개의 코어처럼 인식하게 하여 효율을 높이는 인텔의 기술은?

👁️ 정답 보기 방열판 및 쿨링팬
Q23.Intel CPU 소켓 형태 중 메인보드에 핀이 있고 CPU 바닥면은 평평한 접점으로 되어 있는 방식은?
👁️ 정답 보기 LGA 방식

👁️ 정답 보기 RAM (Random Access Memory)

👁️ 정답 보기 DRAM (동적 램)
Q28.컴퓨터 부팅 시 하드웨어 제어 및 BIOS 시스템이 담겨 있는 읽기 전용 비휘발성 메모리는?
