🌙 야간 다크모드

PC정비사 2급 필기 완전정복

필수 암기 30대 핵심 정리 (내용 무삭제 원본)

1. 윈도우 핵심 실행 명령어 (1과목)
👁️ 내용 보기

• regedit : 레지스트리 편집기

• msconfig : 시스템 구성 (부팅 옵션 및 시작프로그램 제어)

• diskmgmt.msc : 디스크 관리 (파티션 분할, 드라이브 문자 변경)

• devmgmt.msc : 장치 관리자 (하드웨어 충돌 및 드라이버 확인)

2. 리눅스(Linux) 기본 명령어 (1과목)
👁️ 내용 보기

• pwd : 현재 작업 중인 디렉터리 경로 표시

• ls : 디렉터리 내부 파일 목록 보기

• cd : 디렉터리 이동

• chmod : 파일이나 디렉터리의 권한(Permission) 변경

3. CPU 성능 지표 및 캐시 메모리 (2과목)
👁️ 내용 보기

• 클럭(Clock): CPU의 동작 속도 단위 (Hz)

• 캐시 메모리(Cache): CPU와 RAM 사이의 속도 차이를 줄여주는 고속 버퍼 메모리 (SRAM 사용)

• 가상 메모리: RAM 용량이 부족할 때 하드디스크의 일부(pagefile.sys)를 RAM처럼 쓰는 기술

4. 부팅 에러 비프(Beep)음 식별 (3과목)
👁️ 내용 보기

• 삐-, 삐-, 삐- (길게 연속 반복): 메모리(RAM) 장착 불량 또는 접촉 불량

• 삐- 삑, 삑 (길게 한번, 짧게 두세번): 그래픽카드(VGA) 장착 불량 또는 고장

• 삑 (짧게 한번): 정상 부팅 완료

5. 하드디스크 자가 진단 및 구조 (3과목)
👁️ 내용 보기

• S.M.A.R.T.: 하드디스크 고장을 사전에 모니터링하고 진단하는 자체 기술

• 물리적 배드섹터: 디스크 표면(플래터)이 긁혀서 생긴 복구 불가능한 결함

• 논리적 배드섹터: 소프트웨어 오류로 생긴 결함으로, 검사 도구(chkdsk)나 포맷으로 복구 가능

6. 가상 메모리 부족 및 메모리 누수 (3과목)
👁️ 내용 보기

• 메모리 누수(Memory Leak): 프로그램이 종료되었는데도 RAM 자원이 반환되지 않고 묶여 있어 시스템이 점차 느려지는 현상

• 대책: 불필요한 시작프로그램을 줄이거나, 가상 메모리 파일의 크기를 수동으로 확장

7. 악성코드 유형 구분 (3과목)
👁️ 내용 보기

• 랜섬웨어: 파일을 암호화하고 인질극을 벌이며 금전을 요구

• 웜(Worm): 감염 대상 파일 없이도 스스로를 복제하여 네트워크를 마비시킴

• 트로이 목마: 정상 프로그램으로 위장하여 숨어 있다가 정보를 빼냄

8. OSI 7계층 핵심 장비와 특징 (4과목)
👁️ 내용 보기

• 1계층 (물리): 더미 허브, 리피터 (신호 증폭 및 분배)

• 2계층 (데이터링크): 스위칭 허브, 브리지 (MAC 주소 기반 정밀 전송, 데이터 단위: 프레임)

• 3계층 (네트워크): 라우터, L3 스위치 (IP 주소 기반 최적 경로 검색, 데이터 단위: 패킷)

9. 주요 프로토콜과 포트(Port) 번호 (4과목)
👁️ 내용 보기

• SSH (보안 원격접속): 22번

• DNS (도메인을 IP로 변환): 53번

• HTTP (일반 웹): 80번 / HTTPS (보안 웹): 443번

10. IPv4 주소 클래스(Class) 구분 (4과목)
👁️ 내용 보기

• A 클래스: 1 ~ 126 (대규모) / B 클래스: 128 ~ 191 (중규모)

• C 클래스: 192 ~ 223 (소규모, 공유기에서 흔히 쓰는 192.168.x.x 대역)

• 127.0.0.1: 내 컴퓨터 랜카드를 테스트하는 루프백(Loopback) 주소

11. 윈도우 파일 시스템 특징 (1/3과목)
👁️ 내용 보기

• FAT32: 구형 규격, 단일 파일 크기를 최대 4GB까지만 저장 가능

• NTFS: 윈도우 표준 규격, 대용량 파일 저장 가능, 보안 및 압축 기능 지원

• exFAT: 이동식 디스크(USB, SD카드) 전용 규격, FAT32의 4GB 제한을 해결함

12. 레이드(RAID) 디스크 구성 방식 (2과목)
👁️ 내용 보기

• RAID 0 (스트라이핑): 데이터를 나누어 저장하여 속도가 가장 빠름 (하나라도 고장 나면 전멸)

• RAID 1 (미러링): 똑같은 내용을 두 디스크에 동시 저장하여 안정성이 가장 높음 (용량 효율 50%)

• RAID 5: 성능과 안정성을 모두 챙긴 방식, 최소 3개의 디스크 필요, 1개 고장 시 복구 가능

13. 메인보드 CMOS 비밀번호 분실 시 초기화 (3과목)
👁️ 내용 보기

• 방법: 메인보드에 장착된 동전 모양 배터리(CR2032)를 일시적으로 뺐다 꽂거나, 메인보드의 'Clear CMOS' 점퍼 핀을 드라이버 등으로 쇼트(Short)시키면 바이오스 설정과 비밀번호가 공장 초기화됩니다.

14. 파워 서플라이 정격 출력의 의미 (3과목)
👁️ 내용 보기

• 정격 출력: 표기된 최대 순간 출력이 아니라, 컴퓨터가 가동되는 동안 지속적이고 안정적으로 공급할 수 있는 실제 용량를 뜻합니다. 정격 출력이 부족하면 고부하 시 시스템이 자꾸 다운됩니다.

15. SSD 최적화 명령어 'TRIM' (3과목)
👁️ 내용 보기

• TRIM(트림): SSD는 하드디스크와 달리 덮어쓰기가 안 되므로, 파일이 삭제된 공간의 찌꺼기 데이터를 백그라운드에서 미리 지워두는 성능 유지 기술입니다. 구조적 특성상 SSD는 디스크 조각 모음을 수행할 필요가 없습니다.

16. 유선 LAN선(UTP) 규격 및 배열 (4과목)
👁️ 내용 보기

• RJ-45: 유선 랜선 끝에 찍는 플라스틱 커넥터 명칭

• Cat.5e / Cat.6: 기가비트(1Gbps) 인터넷 속도를 지원하는 랜케이블 카테고리 등급

• 다이렉트 케이블: PC와 공유기를 연결할 때 / 크로스 케이블: PC와 PC를 1:1로 직접 연결할 때

17. 무선 LAN(Wi-Fi) 표준 규격 (4과목)
👁️ 내용 보기

• IEEE 802.11n: Wi-Fi 4 (2.4GHz / 5GHz 겸용)

• IEEE 802.11ac: Wi-Fi 5 (5GHz 전용 고속 규격)

• IEEE 802.11ax: Wi-Fi 6 (대용량 다중 접속 최적화 최신 규격)

18. 네트워크 주소 3대 통신 방식 (4과목)
👁️ 내용 보기

• 유니캐스트(Unicast): 가장 일반적인 1 대 1 통신

• 멀티캐스트(Multicast): 특정 그룹을 지정하여 전송하는 1 대 다(Group) 통신

• 브로드캐스트(Broadcast): 동일 네트워크 안의 모든 호스트에게 무조건 뿌리는 1 대 전체 통신

19. IP 자동 할당 프로토콜 DHCP (4과목)
👁️ 내용 보기

• DHCP: 컴퓨터가 켜질 때 공유기(서버)로부터 사용할 수 있는 유동 IP 주소를 자동으로 받아와 세팅해 주는 프로토콜입니다. 만약 이 DHCP 서버에 오류가 생겨 IP를 못 받으면 윈도우는 169.254.x.x라는 임시 주소(APIPA)를 강제로 부여합니다.

20. 서브넷 마스크(Subnet Mask)의 역할 (4과목)
👁️ 내용 보기

• 역할: 주어진 IP 주소에서 '네트워크 ID' 부분과 '호스트 ID' 부분을 구별하기 위해 겹쳐서 사용하는 필터(마스크)입니다. C 클래스의 기본 서브넷 마스크는 255.255.255.0입니다.

21. 윈도우 부팅 및 시스템 복구 '안전 모드' (1과목)
👁️ 내용 보기

• 안전 모드 (Safe Mode): 시스템에 심각한 오류가 생겼거나 드라이버 충돌로 정상 부팅이 안 될 때, 최소한의 기본 드라이버(VGA, 키보드, 마우스)만 로딩하여 부팅하는 진단용 모드입니다. 악성코드를 치료하거나 꼬인 그래픽 드라이버를 삭제할 때 주로 진입합니다.

22. 프로세스와 스레드(Thread)의 개념 (1/2과목)
👁️ 내용 보기

• 프로세스: 메모리에 적재되어 현재 실행 중인 프로그램의 단위

• 스레드(Thread): CPU가 처리하는 실제 작업의 최단 흐름 단위. 1개의 프로세스 내부에 여러 개의 스레드가 동시에 돌아갈 수 있습니다(멀티스레딩).

23. CPU 오버클러킹(Overclocking)과 부작용 (2과목)
👁️ 내용 보기

• 오버클러킹: CPU 제조사가 정해놓은 한계 클럭 속도를 강제로 높여 성능을 끌어올리는 튜닝 기법입니다.

• 부작용: 과도한 발열이 발생하여 시스템이 다운되거나 부품 수명이 급격히 단축되므로, 고성능 사제 쿨러 장착과 전압 미세 조절(언더볼팅 등)이 필수적으로 동반됩니다.

24. 주기억장치 RAM의 종류와 특징 (2과목)
👁️ 내용 보기

• DRAM (동적 RAM): 우리가 흔히 말하는 메인 메모리. 구조가 단순해 대용량으로 만들기 쉽지만, 전원이 켜져 있어도 시간이 지나면 충전된 전하가 방전되므로 주기적인 재충전(Refresh) 회로가 필수적입니다.

• SRAM (정적 RAM): CPU 캐시 메모리에 사용됨. 속도가 매우 빠르고 전원이 켜져 있는 동안 데이터가 유지되나, 구조가 복잡해 가격이 비싸고 대용량화가 어렵습니다.

25. Graphic카드 성능의 핵심 GPU와 VRAM (2과목)
👁️ 내용 보기

• GPU (Graphics Processing Unit): 모니터에 뿌려줄 화면 영상의 복잡한 3D 연산을 전담하는 그래픽카드의 핵심 두뇌 칩셋입니다.

• VRAM (비디오 RAM): 그래픽카드가 연산한 고해상도 이미지 데이터를 임시 보관하는 그래픽 전용 초고속 메모리입니다. 게임 그래픽 옵션을 높일 때 이 VRAM 용량이 많이 소모됩니다.

26. 중앙 처리 장치 메인보드의 '칩셋(Chipset)' 역할 (3과목)
👁️ 내용 보기

• 칩셋: 메인보드의 정중앙에 위치하며 CPU, 메모리, 확장 슬롯(VGA), 저장장치 간의 데이터 흐름(버스 통신)을 총괄 제어하는 가장 중요한 컴퓨터의 교통경찰 역할을 합니다. 메인보드가 지원하는 CPU 종류나 최대 RAM 용량은 이 칩셋 스펙에 의해 결정됩니다.

27. 해킹 기법 구분: 피싱 vs 스미싱 vs 파밍 (3과목)
👁️ 내용 보기

• 피싱 (Phishing): 이메일이나 가짜 링크로 사용자를 속여 비밀번호를 탈취

• 스미싱 (Smishing): 문자 메시지(SMS)에 악성 앱 설치 링크를 보내 소액결제를 유도

• 파밍 (Pharming): 사용자 컴퓨터를 악성코드로 감염시켜, 사용자가 정상적인 포털 주소(예: naver.com)를 쳐도 해커가 만든 가짜 위장 사이트로 강제 접속되게 만드는 고도화된 기법

28. 사설 IP를 공인 IP로 바꾸는 'NAT' 기술 (4과목)
👁️ 내용 보기

• NAT (Network Address Translation): 가정이나 사무실 내부에서 쓰는 수많은 사설 IP 주소들을 인터넷망에서 통하는 단 하나의 '공인 IP' 주소로 변환해 주는 기술입니다. 이 기술 덕분에 인터넷 회선 하나(공인 IP 1개)만 신청해도 공유기를 통해 수십 대의 PC와 스마트폰이 동시에 인터넷을 쓸 수 있습니다.

29. 외부에서 집 PC 서버에 접속하는 '포트 포워딩' (4과목)
👁️ 내용 보기

• 포트 포워딩 (Port Forwarding): 외부 인터넷망에서 집의 공유기(공인 IP)로 특정 포트 번호 신호가 들어왔을 때, 공유기가 이를 내부 사설망의 특정 컴퓨터(예: 192.168.0.5의 80번 포트)로 길을 찾아 바로 연결해 주는 이정표 설정 기술입니다. 집 PC로 개인 홈페이지나 게임 서버를 돌릴 때 반드시 세팅해야 하는 유지보수 기능입니다.

30. 네트워크 경로 추적 및 진단 명령어 (4과목)
👁️ 내용 보기

• ping [IP주소] : 상대방 컴퓨터나 서버가 현재 네트워크에 연결되어 반응하는지 확인 (ICMP 프로토콜 사용)

• ipconfig : 내 컴퓨터의 현재 IP 주소, 서브넷 마스크, 게이트웨이 주소를 확인

• tracert [도메인] : 내 컴퓨터에서 목적지 서버까지 가는 동안 거치는 모든 라우터(경로)의 구간별 속도와 끊기는 지점을 추적하는 도구
